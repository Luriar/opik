## Final Recommendation

1. Treat `outputs/archive/YYYYMMDD/` as the canonical immutable daily artifact.
2. Keep `data/metadata`, master raw/clean OHLCV, master macro, optimized features, and training dataset as production data roots.
3. Move old one-off exports and validation runs to an external long-term archive if disk size matters.
4. Clean cache directories regularly: `.cache`, `.pytest_cache`, and `__pycache__`.
5. Keep empty `notebooks/` unless `src/utils/paths.py` and project structure expectations are changed.
6. Keep `prompts/` for now as implementation provenance; it is safe to omit from deployment packages but useful for development traceability.
